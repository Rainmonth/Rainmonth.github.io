  • 关于 Android RecyclerView 的那些开源 LayoutManager

    提供了一种插拔式的设计，高度解耦，可以设置不同的LayoutManager、ItemDecoration、ItemAnimator、自定义点击事件，自定义长按事件。

  • LayoutManager，用于控制RecyclerView的现实方式（ListView形式、GridView形式和瀑布流形式）；

  • Itemdecoration，用于控制RecyclerView的间隔，即ListView（或GridView）的divider；
  • ItemAnimator，用于控制RecyclerView的Item的动画显示；
  • 自定义事件（长按、点击），未封装好，需要我们可以自己实现；
